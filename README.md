## Usage

### Install dependencies

```bash
yarn
```

### Run dev with hot reloading

```bash
yarn start
```

### Build to `public/`

```bash
yarn build
```

### Serve from `public/`

Local URL is copied to clipboard.

```bash
yarn serve
```
